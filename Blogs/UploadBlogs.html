HTML Tags Reference

Heading Tags

HTML heading tags define content hierarchy, from <h1> (most important) to <h6> (least important).

Main Title

<h1>

Primary page heading, used once per page for optimal SEO.

<h1>Main Title</h1>

Section Heading

<h2>

Major section headings under h1.

<h2>Section Heading</h2>

Sub-section

<h3>

Sub-sections under h2 headings.

<h3>Sub-section</h3>

Sub-subsection

<h4>

Sub-subsections under h3 headings.

<h4>Sub-subsection</h4>
Minor Section
<h5>

Minor sections under h4 headings.

<h5>Minor Section</h5>
Further Detail
<h6>

Further details under h5 headings.

<h6>Further Detail</h6>
SEO Tip: Use a logical heading hierarchy (h1 → h2 → h3) to improve content structure and search engine ranking.

Links & Images

Anchor Tag <a>

Creates hyperlinks to web pages, files, or URLs.

<a href="page.html">Link Text</a>
<a href="page.html" target="_blank">Opens in new tab</a>
<a href="mailto:email@example.com">Send Email</a>
Best Practices
  • Use descriptive link text for clarity.
  • Indicate new tab openings for accessibility.
  • Style visited links for better UX.

Image Tag <img>

Embeds images with required src and alt attributes.

<img src="image.jpg" alt="Description">
<img src="image.jpg" alt="Description" class="img-fluid">
<img src="image.jpg" alt="Description" width="300" height="200">
Accessibility
  • Always use descriptive alt text.
  • Specify width and height to prevent layout shifts.
  • Optimize images for faster loading (e.g., WebP format).

List Tags

HTML lists organize content in unordered, ordered, or description formats.

Unordered List

  • First item
  • Second item
  • Third item 
<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

Ideal for bullet-point lists where order is not critical.

Ordered List

  1. First item
  2. Second item
  3. Third item
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

Used for numbered lists where sequence matters.

Description List

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

Perfect for term/definition pairs.

Table Tags

Tables organize data into rows and columns for clear presentation.

Tag Description Example
<table> Wrapper for all table content. Container
<thead> Contains header rows of the table. Header
<tbody> Contains the main table data. Body
<tr> Defines a table row. Row
<th> Defines a header cell. Header Cell
<td> Defines a standard data cell. Data Cell 
<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Email</th>
      <th>Role</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John Doe</td>
      <td>john@example.com</td>
      <td>Admin</td>
    </tr>
    <tr>
      <td>Jane Smith</td>
      <td>jane@example.com</td>
      <td>User</td>
    </tr>
  </tbody>
</table>
Note: Use tables for tabular data only. For layouts, prefer CSS Grid or Flexbox.
